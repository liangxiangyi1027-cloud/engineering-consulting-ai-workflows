# 估算概算校核规则

## 校核维度

| 维度 | 校核内容 |
| --- | --- |
| 总投资 | 总投资是否与报告正文、附表、汇报材料一致 |
| 工程费用 | 建筑工程、安装工程、设备购置等分类是否完整 |
| 工程建设其他费 | 勘察、设计、监理、建设管理等是否按规则计取 |
| 预备费 | 基本预备费比例是否合理 |
| 建设期利息 | 是否与融资规模、利率、建设期匹配 |
| 资金来源 | 财政资金、专项债、银行贷款、企业自筹等是否闭合 |
| 分年度投资 | 是否与建设时序和资金计划一致 |

## AI 输出要求

自动检查投资表时，应输出：

1. 表内计算错误
2. 表间不一致
3. 与报告文字不一致
4. 明显缺项或重复项
5. 需人工判断的问题
